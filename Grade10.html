IPv4 addresses are split into different classes:

  • Class A: 1.0.0.0 to 126.255.255.255 (for large networks)
  • Class B: 128.0.0.0 to 191.255.255.255 (medium networks)
  • Class C: 192.0.0.0 to 223.255.255.255 (small networks)

üåç IPv4 vs IPv6

  • IPv4: Uses 32-bit addresses (example: 192.168.1.1)
  • IPv6: Uses 128-bit addresses (example: 2001:0db8:85a3::)
  • IPv6 allows more devices to connect and is more secure
