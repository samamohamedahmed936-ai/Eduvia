Grade 5 - How the Internet Works? 🌍

Let’s understand how data travels across the world!

🌐 What is the Internet?

The internet is a giant network of computers all around the world. It's like a big city where computers talk to each other!

📍 What is an IP Address?

Just like your home has an address, every device has a number called an IP address so other computers can find it.

📖 What is DNS?

DNS is like a phone book! It helps turn names like www.eduvia.com into numbers (IP addresses) so your computer knows where to go.

📦 How Data Travels

  • Type a website name
  • DNS finds the address
  • Your device talks to the server
  • The server sends data back to your device
