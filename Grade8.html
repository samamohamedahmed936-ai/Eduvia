Grade 8 - IP Addresses & Data Packets ✉️

Let’s explore how data travels using IP addresses and packets!

📍 What is an IP Address?

It’s like a home address, but for your device! Every computer or phone has an IP address so others can find it online.

📦 What are Data Packets?

When you send a message or load a website, the data is broken into small pieces called packets.

🚚 How Do They Travel?

  • Packets have the sender and receiver IP address.
  • Routers help guide packets to the right place.
  • TCP helps make sure they arrive safely and in order.
